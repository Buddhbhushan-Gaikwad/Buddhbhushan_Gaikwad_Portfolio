# Buddhbhushan Gaikwad — Portfolio

This workspace contains a simple portfolio site (React + Spring Boot) intended for a final-year B.Tech student targeting Java backend / Full-Stack roles.

Structure

- frontend/ — React (Vite)
- backend/ — Spring Boot (Maven)

Quick start (development)

1. Backend
   - cd backend
   - mvn spring-boot:run

2. Frontend
   - cd frontend
   - npm install
   - npm run dev
   - Open http://localhost:5173 (Vite) — the `/api` calls are proxied to the backend at http://localhost:8080

Content notes

All content is honest and student-level. Projects are described in an interview-defendable manner.

Deployment suggestions

- Frontend: Vercel / Netlify
- Backend: Render / Railway
